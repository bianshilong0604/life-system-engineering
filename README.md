# 个人总体设计部 · Life System Engineering

> 把"自我管理"做成一个**可运行、可反馈、可进化**的小系统。  
> 灵感来自钱学森《论系统工程》——把"组织管理"上升为科学方法。  
> 每周 15 分钟复盘,让系统自己变强。

---

## 这是什么

一套基于 markdown 文件 + 纯 Python 脚本的"个人系统工程"框架。它不是待办清单,而是一个会自我改进的闭环:

```
Plan(定) → Execute(做) → Verify(核对) → Reflect(归因) → Patch(提炼规则)
```

- **不靠意志力,靠规则**:把容易栽的坑提炼成可执行规则,下次自动减少
- **少而精**:当前只聚焦 3 个子系统(研究工作 / 学习成长 / 复盘进化),其余封存
- **AI 辅助,人定夺**:LLM 带你跑复盘、沉淀知识,但规则库的最终决定权始终在人手里
- **零依赖、低维护**:纯 Python 标准库,有 Python 就能跑;平时不用管,只在周日复盘时打开

## 核心功能

- **📊 桌面端可视化看板**:浏览器查看子系统 Plan、硬约束、规则库、见闻池
- **📱 手机端 Web UI**:随时记录日记/灵感(💡 标记)、查看规则、筛选见闻池
- **🔄 每周复盘**:AI 辅助四段式复盘(做了啥 → 归因 → 下周计划 → 提炼规则)
- **📝 日记系统**:逐条编辑/删除,灵感用 💡 前缀标记,支持聊天机器人接入
- **🧠 知识沉淀**:视频/文章/灵感自动提炼结构化笔记进知识库
- **🛡️ 规则库**:把踩过的坑提炼成可执行规则(编号 R1/R2...),候选规则池待审批
- **🔍 见闻池**:聚合每日看到的链接/灵感,批量筛选归档或丢弃

## 30 秒上手

```bash
# 1. 克隆
git clone https://github.com/bianshilong0604/life-system-engineering.git
cd life-system-engineering

# 2.(可选)配置 LLM — 想用 AI 功能才需要
cd tools/
cp config.env.example .env
# 编辑 .env 填你的 API Key(任何 OpenAI 兼容提供商都行)
python assistant.py test   # 测连通

# 3. 启动
python server.py           # 桌面端看板(自动开浏览器 localhost:8770)
python mobile_ui.py        # 手机端 UI(访问 <你的服务器IP>:8780/mobile)
```

**不想用 AI?** 跳过第 2 步,纯手动用 —— 每周打开 `02_每周复盘/` 新建一篇,照着四段式手写即可。

详细见 [系统说明](系统说明.md) 和 [LLM配置说明](LLM配置说明.md)。

## 谁适合用

- 觉得"自我管理工具"总是用着用着就放弃的人
- 想把碎片学习/灵感真正沉淀下来、而不只是收藏的人
- 一个人推进一个主线项目(研究/写作/创业)、需要守住精力边界的人
- 相信"系统比意志力可靠"的人

## 核心理念(摘自总纲领)

**三个核心问题(每次复盘必问):**
1. 这次搞砸是**能力问题**还是**系统问题**?
2. 系统问题归类:目标不清 / 时间没留 / 工具差 / 边界乱 / 反馈少 / 标准不清?
3. 有没有**一条规则**能让同类问题下次自动减少?

**防过度系统化(最重要):**
- 指标少到一只手数得过来,记录花的时间不能超过它省的时间
- 关系、创造力、当下的快乐 —— 不该被量化
- **如果这套东西让你更累更焦虑,就是搭错了,立刻简化**

## 快速上手示例

### 手机端日记(随时记录)
```bash
python tools/mobile_ui.py
# 访问 http://<你的IP>:8780/mobile
# - 点「写」tab,输入日记(灵感开头加💡)
# - 点「看板」查看硬约束/规则/子系统
# - 点「见闻池」筛选今日链接归档/丢弃
```

### 桌面端复盘(每周一次)
```bash
python tools/server.py
# 自动打开浏览器 localhost:8770
# - 点「本周复盘」→ AI 带你走四段式
# - 复盘产出的新规则 → 规则库_候选.md
# - 周日确认无误 → 手动搬进规则库.md
```

### CLI 快速操作
```bash
# AI 辅助复盘
python tools/assistant.py review

# 记录踩坑 → AI 判断是否该入规则库
python tools/assistant.py pit "今天又忘了提前确认会议室"

# 沉淀知识(视频/文章/灵感)
python tools/assistant.py learn "https://example.com/article"

# 月度体检(AI 看系统该保持/解封/简化)
python tools/assistant.py checkup
```

## 技术栈

- **语言**:纯 Python 3.8+(标准库,零 pip 依赖)
- **前端**:原生 HTML/CSS/JS(无构建工具,双击打开即用)
- **数据**:Markdown + JSON(纯文本,永远可读,git 友好)
- **AI 可选**:兼容任何 OpenAI API 格式的 LLM(Claude/GPT/本地模型)
- **部署**:单机 Python HTTP Server(无需 Docker/数据库/云服务)
