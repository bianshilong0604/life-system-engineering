# 03_每日日记 · 原始流水(只追加,不提炼)

每天随手记「做了什么 / 收获 / 小想法」,周末喂给复盘 / AI 周整理。
日记是**复盘的原料仓** —— 平时零成本追加,需要时才提炼。

## 铁律(防过度系统化)

- **日记 = 原始流水**:只追加,不提炼,不调 LLM。
- **提炼只在周整理 / 复盘**:LLM 读日记产出归类,不回写原始日记。
- **低摩擦**:一句话也能记;不打卡、不强制、不评分、不计数。
- **append-only**:日记只追加不会腐烂,无需新鲜度检查。

## 文件格式

一天一文件:`YYYY-MM-DD.md`,首行标题,每条带 `HH:MM` 时间戳:

```text
# 2026-06-17 日记

- 09:30 今天把规则库改成逐条折叠了,和硬约束对齐
- 14:00 PhysGuard 卡在数据加载,查了半天是路径编码
- 22:10 想法:日记也许该自动喂给复盘
```

一天可多条,网页 / CLI 都往同一文件**追加**(不覆盖)。

> 同目录的 `_示例_格式.md` 是格式演示(不以日期开头,不会被看板当真实日记、也不会被 gitignore)。
> 你自己的日记文件名以日期开头(`2026-*.md`),会被 .gitignore 忽略,留在本地。

## 用法

**CLI(纯文本,不需要 API key):**

```bash
python tools/assistant.py diary           # 记一笔(追加到今天)
python tools/assistant.py diary_digest    # 用 LLM 整理本周日记
```

**网页看板:**

启动 `python tools/server.py`,在「📓 每日日记」卡片里:

- 内联 textarea + 「✏ 记一笔」→ 追加到今天
- 点日期行 → 折叠展开当天全文
- 「🔍 用 AI 整理本周日记」→ LLM 归类本周日记,存成 `_周整理_YYYY-WW.md`

## 周整理产物

`_周整理_YYYY-WW.md`(下划线前缀 → 不被 `glob("20*.md")` 当日记,不污染列表)。

> 本目录是开源模板占位 —— 示例文件 `2026-06-17.md` 仅为格式演示,不含真实内容。
> 你自己的日记留在本地,不要提交到公开仓库(已建议在 `.gitignore` 里忽略真实日记)。
