# 子系统 · 学习成长

> 把"读到的/看过的"变成"用得上的"。核心动作是**结构化沉淀**，不是收藏。

## 职责
- 把碎片信息（文章/视频/灵感）结构化进知识库
- 每条沉淀都标"今日意义"——能用到我哪件具体的事上
- 防"囤积症"：判断每条是"值得长期留存"还是"看完就够"

## 健康信号
- [ ] 本周沉淀了几条（知识库 [学习成长_知识库/](学习成长_知识库/)）
- [ ] 有没有"回头翻就能找到"（检索 < 翻找）
- [ ] 沉淀的有没有真的被用到（用不上的要敢于丢弃）

## 工作流（系统已内置）
1. **今日见闻池**（[03_今日见闻/](../03_今日见闻/)）：先入池，不急着判断
2. **每晚筛选**：看板里选「归档」或「丢弃」
3. **归档**：AI 自动结构化（一句话核心 / 关键点 / 今日意义 / `[[双链]]` 关联）

## 示例沉淀
见 [学习成长_知识库/](学习成长_知识库/) 里的示例笔记，模仿其结构即可。
